Polish Weather API

Querry examples:

Access all data: http://127.0.0.1:5000/weather

Date endpoints:

Access data from exact date: http://127.0.0.1:5000/weather/date/2019-01-01

Access data from date range: http://127.0.0.1:5000/weather/date/between/2005-01-01/2010-01-02

Access data after: http://127.0.0.1:5000/weather/date/after/2005-01-01

Access data before: http://127.0.0.1:5000/weather/date/before/2010-01-02
